# 음정 정거장 v2.0 구현 계획

마스터 프롬프트의 STEP 00~17을 현재 저장소의 실제 산출물로 압축해 실행한다.

1. 데이터와 순수 엔진: 25칸 보드, 12개 음정, 48장 카드, 초록·노랑·빨강 계산.
2. 상태·저장·오디오: reducer 중심 상태 흐름, 안전 checkpoint, Web Audio/Null adapter.
3. 화면 셸: 타이틀, 노선도, 게임, 수첩, 결과, 설정.
4. 학습 루프: 입문·기본·완주·청음·통합 노선, 오답 교정, 수첩·별·잠금.
5. 품질: strict TypeScript, lint, 단위 테스트, 추적성, 정적 PWA shell.
6. 브라우저 검증: 360/390/430 폭, 키보드, 모바일 overflow, production build.

원격 배포·푸시·외부 메시지 발송은 이 작업 범위에 포함하지 않는다.
